## NOTAS PARA EL EDITOR

### Estilo visual
- **Paleta:** Azul `#1E40AF` + dorado `#F59E0B` + blanco. Sin otros colores.
- **Tipografía en pantalla:** Bricolage Grotesque (Bold) para títulos, Inter para subtítulos.
- **Ritmo:** Cortes rápidos en la escena 1 (el caos). Más respirado desde la escena 3.
- **Música:** Instrumental. Warm, moderno, progresivo. Nada de stock genérico. Algo que suene a propósito y comunidad.
- **No usar:** Imágenes stock de gente orando con las manos levantadas. Demasiado usado, resta credibilidad.

### Screen recordings necesarios
Conseguir con el equipo de JETRO:
1. Dashboard mobile — pantalla de bienvenida con stats flotando.
2. Jerarquía de liderazgo — árbol navegable.
3. Registro de asistencia — el toggle verde/rojo de cada miembro.
4. Aprobación de reporte — el flujo de un supervisor aprobando.
5. Mapa de zonas con pines.
6. Gráfico de crecimiento (panel de reportes).
7. Cards de módulos.

### Formatos de entrega
- MP4 1920×1080 (para YouTube/web)
- MP4 1080×1920 vertical (para Instagram Reels / TikTok)
- Capítulo de 15s para el anuncio (escenas 3 + 8 comprimidas)
